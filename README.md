# My AI Prompting Framework

## Explain Prompt
When we want to understand the concept , we should use Explain Prompt
eg:BAD PROMPT:add AI prompting framework
GOOD PROMPT:Explain JavaScript variables to a beginner using simple examples and analogies.

## Pair Programmer Prompt
Used when I am coding and want guidance.
eg: BAD PROMPT:Build a to-do app in JavaScript
GOOD PROMPT:“Act as my senior developer.
I want to build a simple to-do app in JavaScript.
Don’t write full code.
Guide me step by step and ask me what to write next.”

## Debug Prompt
Used when code breaks and I want to learn why.
eg: BAD PROMPT:My code is not working, fix it”
GOOD PROMPT:“Here is my code.
Do not rewrite it.
Explain why it’s not working and what I should check.”

## Refactor Prompt
Used when code works but needs improvement.
eg: BAD PROMPT:Make my code better”
GOOD PROMPT:My code works correctly.
How can I improve readability and structure without changing the logic?”
